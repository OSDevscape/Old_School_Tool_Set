# Old School Tool Set (OSTS) v10.0.0

A single‑page, mobile‑first Old School RuneScape sidekick that runs entirely in your browser.  
Track your account, bossing, timers, and Slayer bestiary in one sleek, PWA‑ready dashboard. 

---

## ✨ Feature Overview

- **Account overview**
  - Pulls live Wise Old Man data for levels, XP, rank, and combat snapshot.
  - Shows next 99, highest skill, and maxed‑skill count at a glance.

- **Skills**
  - Responsive skills grid with per‑skill level, XP, rank, and progress bars.
  - Tap for a detailed sheet with stats, descriptions, and progress charts. 

- **Bossing**
  - Boss summary metrics (bosses with KC, total KC, EHB, best rank).
  - Expandable boss cards with KC, rank, EHB and “boss state” subpanels.

- **Gains**
  - Day / Week / Month / Year XP gain views.
  - Inline charts powered by Chart.js for quick visual progress checks. 

- **Timers**
  - Herb, tree, patches, birdhouses and more, all mapped to in‑game durations.
  - Pin favourites, track live countdowns, and reset/clear via the timer tools sheet. 

- **Bestiary**
  - Slayer‑centric monster browser with filters by name, location, weakness, notes and group.
  - Variant / superior / boss tabs, assigned‑head icons, stats, drops and wiki links.

- **News**
  - Inline card feed of official OSRS news posts with thumbnails and tags.
  - “Open Official OSRS News” CTA to jump out to the main site.
