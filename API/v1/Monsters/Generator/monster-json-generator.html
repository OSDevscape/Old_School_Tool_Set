Paste OSRS wiki-style text and generate the generic entity/form/spawn/task/loot schema.

Mobile-first Schema-aware No server needed

How it maps

  • Top monster article becomes entity, statProfiles, common forms, spawns, combatNotes, and lootTables.
  • Slayer task text becomes encounters and taskData.
  • Level/location variations become separate forms linked to reusable spawnIds.
  • Sections with incomplete data stay valid by preserving raw strings in notes instead of discarding them.
